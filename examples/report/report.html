Background

Why HTML for print

HTML and CSS are an underrated typesetting engine. With the Paged Media specification, the same markup that renders in a browser can be paginated into a print-ready document — complete with page boxes, running headers, and cross-references — without opening a layout application.

The document is the source. The PDF is just one of its renderings.

That matters when the content is generated or frequently revised: a report built from data, a localized manual produced in twelve languages, or an ebook regenerated on every edit. Manual layout doesn't scale to those; a pipeline does.

Mechanism

How the pipeline works

The tool performs four deterministic steps and leaves the document's own design untouched:

Pipeline stages
StageWhat happens
1. ReadLoad the source HTML file.
2. InlineEmbed local images as data URIs so the file is portable.
3. OverlayInject print CSS: page size, margins, folios, live TOC.
4. RenderPaginate with Paged.js (headless Chrome) and write the PDF.

Because rendering uses Chrome's engine, vector text and SVG are preserved and fonts are embedded by the PDF writer.

Output

What the overlay adds

Page numbers

Folios appear from the imprint page onward; the cover is excluded. Position, color, font, and size are configurable.

A live table of contents

Each TOC link is resolved to its real page number after layout with target-counter() — no manual updates when the content shifts.

Pagination guards

Headings stay with the text they introduce; figures, tables, and quotes are kept whole rather than split across a page boundary.

Caveats

Notes & trade-offs

CSS Grid does not fragment across pages — a grid block jumps whole to the next page and can strand a heading. Convert long grids to multi-column layout when they need to flow. Web-font headings render correctly but are not extractable as text, so verify content against the body text and a page image rather than the headline.

The renderer is the open-source Paged.js, so the entire pipeline can be handed to a client with no licensing cost and no watermark.
